Web 实时通讯技术介绍

Web Real-Time Communication， WebRTC，使浏览器可以直接和其他浏览器交互，从而形成了包括三角模式和梯形模式在内的多种体系架构。其中，他的多媒体功能非常先进。WebRTC旨在将实时通信功能引入到所有浏览器中，并通过标准的[HTML5]标签和JavaScriptAPI，使这些功能可为Web开发者所用。
